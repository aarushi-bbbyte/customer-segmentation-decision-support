## Dataset

This project uses the **Online Retail Dataset** from the UCI Machine Learning Repository.

Due to file size constraints, the dataset is not included in this repository.

### Download Instructions
1. Visit: https://archive.ics.uci.edu/ml/datasets/online+retail
2. Download: `Online Retail.xlsx`
3. Convert it to CSV
4. Place it in this folder as: `online_retail.csv`
